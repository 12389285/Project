Number of terrorist attacks

In the past 50 years there have been a lot of attacks arround the world. The visualization below shows the number of attacks global and per country. When using the slider the data updates by year. By selecting a country the barchart will update with the number of attacks in the selected country and year, this way it is possible to compare data in different years. Also by selecting a country the line chart below will update with the data of the fatal and non fatal incidents. In some cases there is no data available for that year or country.
